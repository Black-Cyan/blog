## 📄主要文件说明

[**friends.json**](../friends.json)  

其作为存储友链的 JSON 文件，供博客网站读取渲染友情链接界面。若你想跟我交换友链，请按照以下格式在该仓库提交 **issue** 或者 **Pull Request**:  

你好！我是xxx，想与你交换友情链接，我的邮箱是 \[xxx@xxx.xxx\]\(mailto:xxx@xxx.xxx\)，以下是我的网站的详细信息：

```json
{
    "name": "你想展示的昵称",
    "desc": "你的网站的简短描述",
    "link": "你的博客网站地址",
    "avatarUrl": "你的头像地址",
    "avatarAlt": "你的头像提示信息(可为空)"
}
```

你还可以在此处补充更多信息。  

我收到后会通过邮箱联系你，但可能不会很快。
