## 📄 Main File Description

[**friends.json**](friends.json)

It serves as a JSON file for storing friend links, which is read by the blog website to render the friend links page. If you want to exchange friend links with me, please submit an **issue** or **Pull Request** in this repository according to the following format:

Hello! I'm xxx and would like to exchange friend links with you. My email is \[xxx@xxx.xxx\]\(mailto:xxx@xxx.xxx\). Here are the details of my website:

```json
{
    "name": "The nickname you want to display",
    "desc": "A brief description of your website",
    "link": "The URL of your website",
    "avatarUrl": "The URL of your avatar",
    "avatarAlt": "Prompt information for your avatar (can be empty)"
}
```

You can also add more information here.

I will contact you via email after receiving it, but it may not be very soon.
